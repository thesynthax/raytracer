# Raytracing on Rust
Learning Rust while building a project on it. 
Raytracing is an intriguing concept, and this is my attempt to learn Rust while understanding the concepts of raytracing first-hand through code.

# Current Status:
- Core raytracer completed.
- Takes hours, or even days to compute a relatively detailed image.
- Have to implement parallelisation to use all the threads.
- Can implement CUDA Programming to use the GPU instead of CPU.
