# Raytracing on Rust
Learning Rust while building a project on it. 
Raytracing is an intriguing concept, and this is my attempt to learn Rust while understanding the concepts of raytracing first-hand through code.

### Current Status:
    - Core raytracer completed
    - Spheres, Axis-aligned Boxes and Rectangles implemented.
    - Diffuse, Metallic, Refractive and Emissive materials implemented.
    - Bounding boxes for each hittable implemented.
    
### Near goals:
    - To implement rotation of objects (non axis-aligned).
    - To implement ability to import textures.
    - To implement ability to import 3D models (obj, fbx etc).
    - To implement random scene generation, and programmable scene generation.
    
### Issues:
    - Takes hours, or even days to compute a relatively detailed image.
    
### Ambitious Goals:
    - Have to implement parallelisation to use all the threads.
    - Can implement CUDA Programming to use the GPU instead of CPU.
